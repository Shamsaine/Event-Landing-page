# Event-Landing-page

Alteturia landing site organized for scalable multi-page growth.

## Structure

- `index.html` - Home page entry point
- `pages/about.html` - About page
- `pages/events.html` - Events page
- `components/navbar.html` - Shared navigation
- `components/footer.html` - Shared footer
- `assets/css/styles.css` - Global styles
- `assets/js/script.js` - Shared behavior and data
- `logo-preview/` - Brand logo assets

## Notes

- The canonical event page is `pages/events.html`.
- Shared components are loaded dynamically so the layout stays consistent as the site grows.
